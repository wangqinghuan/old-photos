大约摄于1897年，一位携带行囊的游牧说书人。

挪威讲故事的人艾利夫·布拉泰内（Eiliv Braatene）一生大部分时间都在流浪，他所有的家当仅装在一个锡罐和一个小包裹里。尽管他身体残疾，身高不足一米，还患有足疾，但他却活出了真正旅行者的风采。他脑中装着海量的童话故事，凭借出色的叙事才华，他成了各大农场受欢迎的座上宾，尤其深得孩子们的喜爱。1
