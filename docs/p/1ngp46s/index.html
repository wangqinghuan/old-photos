罗莎莉·简·威利斯（Rosalie Jean Willis）于1937年1月28日出生在西弗吉尼亚州的本伍德，她是克拉伦斯·“牛仔”·威利斯（Clarence "Cowboy" Willis）及其妻子弗吉尼亚·麦克宁奇（Virginia McNinch）的女儿，这对夫妇于1929年在西弗吉尼亚州的马歇尔县结婚。罗莎莉是三个姐妹中最小的一个，上面还有艾琳（Eileen）和艾米（Amy）；他们还有一个兄弟叫克拉伦斯·“巴斯特”·威利斯（Clarence "Buster" Willis）。“牛仔”和弗吉尼亚在孩子们还小的时候就分开了。 1954年，查尔斯·曼森（Charles Manson）和他的母亲凯瑟琳（Kathleen）住在西弗吉尼亚州的查尔斯顿或附近，那时他与罗莎莉的父亲“牛仔”成了朋友。正是通过这层关系，他认识了罗莎莉；他们于1955年1月17日在西弗吉尼亚州马歇尔县的拿撒勒教堂结婚；当时她“17岁”。选这个教堂是因为查尔斯的祖母南希·英格拉姆（Nancy Ingram）是那里的教友。与此同时，凯瑟琳决定搬到洛杉矶地区。 查尔斯和罗莎莉决定去洛杉矶探望他的母亲，或者干脆住在那儿。为了这趟旅程，他偷了一辆车。他们到加州没多久，他就因为偷车被捕了，所以1956年4月10日他们在加州洛杉矶县（CADI）唯一的孩子小查尔斯·米尔斯·曼森（Charles Milles Manson Jr.）出生时，他并不在场。 凯瑟琳和罗莎莉确实去监狱探望过查理，但罗莎莉很快就厌倦了这种生活，转而跟一个叫杰克·怀特（Jack White）的男人在一起了。她为这个男人生了两个儿子：1958年2月3日在洛杉矶县出生的杰西·J·怀特（Jesse J White，CABI），以及1959年4月4日在河滨县出生的杰德·怀特（Jed White，CABI）。在此期间，她于1957年7月9日向查理提出离婚，最终判决于1958年9月10日生效。 显然，她在那段时间嫁给了杰克·怀特，他们后来于1965年离婚。 1971年1月3日，她11岁的儿子杰德·怀特在一场意外枪击中丧生。 1971/72年间，这家人住在马萨诸塞州，她儿子查尔斯·曼森（小曼森）就是在那里拿到的社保卡。 1986年8月23日，她的儿子杰西·怀特在德克萨斯州休斯顿因吸毒过量死亡。1993年6月29日，她的儿子查尔斯·曼森（小曼森）开枪自杀。 她最后一任丈夫是沃伦·霍华德·“杰克”·汉德利（Warren Howard "Jack" Handley），他于1926年4月1日在印第安纳州出生，1998年11月10日在洛杉矶去世，但生前住在内华达州克拉克县的拉斯维加斯。 她于2009年8月21日在亚利桑那州图森市因肺癌去世。她和沃伦都安葬在内华达州博尔德城的南内华达退伍军人纪念公墓。
